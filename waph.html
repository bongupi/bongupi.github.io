LAB-1: In this lab, we explored the world of computer communication through protocols, with a focus on HTTP. Moving forward, we delved into web application programming, creating CGI scripts in C, HTML, and PHP. Deploying these scripts allowed us to access dynamic web pages through browsers. Wireshark remained a valuable tool for analyzing the specifics of HTTP GET and POST requests, offering insights into the communication flow between clients and servers.
